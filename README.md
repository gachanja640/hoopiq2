# HoopIQ 🏀
**Basketball Intelligence — runs 100% free on GitHub Pages**

No API key. No backend. No cost. Powered by ESPN's free public data.

## Deploy in 3 steps

**1. Push to GitHub**
```bash
git init && git add . && git commit -m "HoopIQ"
git remote add origin https://github.com/YOUR_USERNAME/hoopiq-app.git
git push -u origin main
```

**2. Enable GitHub Pages**  
Settings → Pages → Source → **GitHub Actions** → Save

**3. Done ✅**  
Live at `https://YOUR_USERNAME.github.io/hoopiq-app/`

No secrets or API keys needed.
